# DeepSORT

The "q" button quits the video.\
The "space" button stops/resume the video.\
The "r" button moves to the first frame of the video.\
The "a" button moves 30 frames back.\
The "d" button moves 30 frames forward.

config.py file contains constants parameters used in algorithms.

Example run of algorithm: \
`python Main.py TestVideos/MOT16-09-raw.webm output_file_name`

File output file saves after quit the visualization.
Text representation saves to the json file with same name like second parameter.
